# Tool Learning Log

## Tool: **A-Frame**

---

### 3/16/26:
* `a-scene` is used as the container for the 3d world
* You can add objects using `a-entity`. These objects include shapes, models, lights, and cameras.
* You can insert primitive shapes using `geometry="primitive: box/sphere/cylinder"` for simple 3d objects.
* You can set position, rotation, and scale using `position "x y z"` to move size and objects.
* You can add clicks, animation, and movement with A-Frame components like `animation` or `cursor`.
